# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

# Group2-JemmiaDiamondShopWebsiteProject

-Project Description
Jemmia is an e-commerce website specializing in selling diamonds and high-end jewelry. Users can register, log in securely, search for products, add items to their cart, and purchase online. The website supports a Wishlist feature to save favorite products. The system ensures user information security through authentication and permission management mechanisms. Administrators can manage users, products, and orders. The project aims to build a modern, convenient, and secure online shopping platform.

-Insallzation guide
The project is built with a backend using Node.js and Express.js, a frontend using React combined with Vite, and a MongoDB database. To install, first install Node.js and MongoDB on your machine. Then clone the source code to your machine and run `npm install` for both the backend and frontend directories. Next, configure the .env file to connect to the database and the necessary environment variables. Start the backend using `npm start` or `npm run dev`, and the frontend using `npm run dev`. After successful execution, access the application via your browser at localhost.

Frontend : react+vite
